Privacy Policy for AI Side Tools

Effective date: 16 May 2026

AI Side Tools is a browser extension that adds a floating side panel to ChatGPT for managing reusable prompts, snippets and workflow tools.

Data stored by the extension

AI Side Tools stores the tools, prompt templates, categories, favourites and settings created by the user using Chrome local storage. This data is stored locally in the user's browser.

Data collection

AI Side Tools does not collect, sell, share or transfer personal data to the developer or to any third party.

Network requests

AI Side Tools does not send user content to external servers. The extension does not use external servers, analytics, tracking scripts or advertising tools.

How the extension works

AI Side Tools only inserts user-created text or prompt templates into the ChatGPT input field on chatgpt.com.

User control

Users can add, edit, delete, export and import their tools from the extension panel. Users can remove locally stored extension data by uninstalling the extension or clearing the extension's local storage in Chrome.

Third-party services

AI Side Tools is designed to work with ChatGPT, but it is an independent browser extension and is not affiliated with OpenAI.
